# Changelog

All notable changes to Slide Media Gallery will be documented in this file.

## [1.0.0] - 2024-11-24

### Added
- Initial release of Slide Media Gallery plugin
- Custom post type for photo albums
- Album management interface with drag-and-drop ordering
- Support for adding titles and descriptions to photos
- WordPress Media Library integration
- Google Drive image support
- Multiple display layouts: Grid, Slider, and Masonry
- Responsive design for all device sizes
- Built-in lightbox for viewing full-size images
- Shortcode support for embedding albums: `[smg_album]`
- Configurable columns for grid layout
- Admin interface with intuitive controls
- Security: URL sanitization and input validation
- Performance: Conditional script loading

### Features
- ✅ Create and manage multiple photo albums
- ✅ Drag-and-drop to arrange photos
- ✅ Add title and description for each photo
- ✅ Support for both Media Library and Google Drive
- ✅ Three layout options (grid, slider, masonry)
- ✅ Mobile-responsive design
- ✅ Lightbox gallery viewer
- ✅ Easy shortcode integration

### Technical Details
- WordPress 5.0+ compatible
- PHP 7.0+ required
- Uses jQuery UI Sortable for drag-and-drop
- No external dependencies
- Fully translatable (text domain: slide-media-gallery)
